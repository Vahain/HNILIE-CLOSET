Mở file discounts.txt

Cách thêm mã giảm giá

Mở file assets/discounts.txt bằng Notepad và thêm dòng theo định dạng:

# Dòng bắt đầu bằng # là ghi chú, bị bỏ qua
TÊN_MÃ | % | 20 | Mô tả giảm 20%
TÊN_MÃ | vnd | 50000 | Mô tả giảm 50.000đ
% = Giảm theo phần trăm (10%-60%)
vnd = Giảm số tiền cố định (5k, 10k, 100k...)
Lưu file → refresh trang là xong
